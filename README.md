# Конвертер валют (Currency Converter)

**Автор:** Аксенов Данил Денисович

---

## Описание

Десктопное GUI-приложение для конвертации валют с использованием внешнего API.
Программа получает актуальные курсы валют, выполняет конвертацию и сохраняет историю операций.

**Используемый API:** [ExchangeRate-API](https://www.exchangerate-api.com/)

---

## Функциональные возможности

- ✅ Получение актуальных курсов валют из внешнего API
- ✅ Конвертация между любыми поддерживаемыми валютами
- ✅ Проверка корректности ввода (сумма > 0)
- ✅ История всех конвертаций (дата, сумма, результат, курс)
- ✅ Автоматическое сохранение истории в JSON
- ✅ Очистка истории
- ✅ Экспорт истории в отдельный JSON-файл
- ✅ Демо-режим при отсутствии интернета или API-ключа

---

## Как получить API-ключ

1. Перейдите на сайт [ExchangeRate-API](https://app.exchangerate-api.com/sign-up)
2. Зарегистрируйтесь (бесплатный тариф позволяет до 1500 запросов в месяц)
3. Подтвердите email
4. Скопируйте ваш API-ключ с дашборда
